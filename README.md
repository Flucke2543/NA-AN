# NA-AN
1. ไฟล์ index.html (โครงสร้างกระดูก)
ไฟล์นี้คือการบอกว่าบนหน้าเว็บ "มีอะไรอยู่บ้าง" แต่ยังไม่ได้บอกว่า "สวยแค่ไหน"
<nav class="navbar">: แถบเมนูด้านบนสุด (NA-AN / ชั้นหนังสือ / เกี่ยวกับเรา)
<header class="hero">: ส่วนหัวกว้างๆ ที่มีรูปพื้นหลังเท่ๆ กับคำโปรย "เพราะวรรณกรรมโลก..."
<section id="store" class="store">: พื้นที่หลักของร้านค้า
.book-grid: กล่องใบใหญ่ที่เอาไว้ "ขัง" การ์ดหนังสือทั้งหมดให้เรียงกัน
.card-link: (แท็ก <a>) คือจุดที่มึงต้องใส่ Link PDF ใน href และมันยังทำหน้าที่เป็น "พื้นที่คลิก" ของการ์ดทั้งใบ
.book-card: ตัวการ์ดหนังสือหนึ่งเล่ม ประกอบด้วย รูปปก (.cover-box), ชื่อเรื่อง (<h3>), ชื่อคนเขียน (<p>), และสถานะ (.card-footer)

2. ไฟล์ style.css (การตกแต่ง/เมคอัพ) นี่คือจุดที่มึงต้องมา "ตบ" ให้มันสวย จำชื่อคลาส (ที่ขึ้นต้นด้วยจุด .) ให้ดี:
:root: คือ "ถังสี" อยากเปลี่ยนธีมเว็บจาก ทอง-แดง เป็น เขียว-ฟ้า ก็มาแก้ Code สีที่นี่ที่เดียวจบ (เช่น --gold, --bg)
.book-grid:
- grid-template-columns: repeat(3, 1fr); คือการสั่งให้ เรียง 3 คอลัมน์ ถ้าอยากให้เรียง 4 ก็เปลี่ยนเลข 3 เป็นเลข 4
.book-card:
- background: สีพื้นหลังการ์ด
- padding: ระยะห่างข้างในการ์ด (เพิ่มเลข = การ์ดดูโปร่ง, ลดเลข = การ์ดดูกะทัดรัด)
- transition: ความเร็วตอนที่เมาส์ชี้แล้วมันเด้งขึ้นมา
.cover-box img:
- height: ความสูงของรูปปก (ถ้าอยากให้การ์ดสั้นลง ก็มาลดเลขที่นี่)
- object-fit: cover;: สั่งให้รูป "ครอป" ให้พอดีกล่อง ไม่ให้รูปเบี้ยวหรือยืด
.card-link:
- text-decoration: none;: ลบขีดเส้นใต้ลิงก์ที่น่ารำคาญออก
- color: inherit;: สั่งให้มันใช้สีตามที่เราตั้งไว้ ไม่ให้บราวเซอร์เปลี่ยนเป็นสีม่วง
.card-footer:
- display: flex; justify-content: flex-end;: คำสั่งที่เพิ่งทำสำเร็จ คือการผลักลูก (คำว่าอ่านฟรี) ไปไว้ขวาสุด
3. ส่วนของ Reader (ถ้ายังเก็บไว้)
.reading-container: ตัวคุมความกว้างของเนื้อหา (ตั้งไว้ 700px เพื่อให้สายตาไม่ต้องกวาดไกลเวลาอ่าน)
.first-letter: (ที่สั่งลบ) เอาไว้ทำตัวอักษรตัวแรกให้ใหญ่ๆ เหมือนนิยายฝรั่ง
